Dados Principais
Os Dados Principais devem ser obtidos preferencialmente utilizando os IDs dos embarques. Essa abordagem garante a precisão do valor total final das peças, pois cobre todas as informações relevantes do embarque.

Também é possível buscar diretamente das CAFs. No entanto, essa abordagem pode impedir o cálculo correto do valor total final das peças.

**Importante**: Se você utilizar apenas os Dados Principais, pode não capturar todas as peças, pois este método pode não incluir as injeções e a coleta completa. Para garantir a inclusão dessas informações, é necessário utilizar também os Dados CAF.

Exemplo de Dados Principais:
AWB ( Exemplo: AMZB281963650tx ).
Rota ( Exemplo: 70-FOR-CE-INT-[V08] ).
ID do Embarque ( Exemplo: 3108261 ).
ID da CAF ( Exemplo: 16041584 ).
Descrição do Status ( Exemplo: EM ROTA ).
Nome do Motorista ( Exemplo: SERGIO PINHEIRO DE SA RORIZ ).
Dados CAF
Para obter os Dados CAF, você deve copiar as CAFs do Dados Principais e realizar uma nova busca. Apenas com essa abordagem você conseguirá obter as informações corretas, incluindo as injeções e coleta, pois somente os dados do Dados Principais não fornecem todas as informações necessárias para uma coleta completa.

Exemplo de Dados CAF:
AWB ( Exemplo: AMZB281963650tx ).
Rota ( Exemplo: 70-FOR-CE-INT-[V08] ).
ID do Embarque ( Exemplo: 3108261 ).
ID da CAF ( Exemplo: 16041584 ).
Descrição do Status ( Exemplo: EM ROTA ).
Nome do Motorista ( Exemplo: SERGIO PINHEIRO DE SA RORIZ ).
Dados dos Tickets
Para os Dados dos Tickets, você deve coletar:

AWB ( Exemplo: AMZB281963650tx ).
Descrição do Ticket ( Exemplo: Encomenda auditada - 5 : AMZB281963650tx ).

Observação: Para baixar o relatório de tickets no ICS, certifique-se de marcar a opção Rota em Dados da encomenda e, em seguida, selecione a caixa Gerar Relatório de Tickets (essa opção fica ao final da página).
Excluir Motoristas
Se você deseja apagar informações de motoristas, utilize a opção disponível para essa ação. Certifique-se de que os dados estão corretos antes de realizar a exclusão.
Baixar Imagem
Ao baixar a imagem do relatório, ela também será salva na sua área de transferência em formato Excel. Certifique-se de verificar e garantir que o relatório seja gerado corretamente.
