Você está com duvidas??
Quais dados devo colocar?
Nome ( nome de quem esta fazendo o relatório ).

Horário ( referente a hora em que o caminhão chegou a base, essa informação está na foto tirada do caminhão ).

Data chegou ( refere-se a data que o caminhao chegou a base, essa informação está na foto tirada do caminhao ).

Data prioridade ( a data da prioridade é referente as encomendas que estão mais perto do prazo, normalmente no dia seguinte a Data chegou).

Coleta STD/Coleta EXP ( é o número da coleta encontrada no ICS que é usado para baixar o relatório).

VRID/Qntd folha ( essas informações ficam anexadas a folha entregue pelo motorista, e são usadas para gerar as mensagens).
Quais dados devo colocar na coleta?
Rotas ( Rota da encomenda ex: 74-AJU-SE-LOC-[30A] ).

Tipo ( é a prioridade da entrega do produto, podendo variar de '0' STD, '6' EXP ).

Data ( data de promessa de entrega, a data prevista para a entregar o produto. ).

IMPORTANTE

Ao copiar os dados ( direto da planilha ), nao deve haver nenhum dado que não seja os mencionados a cima, ( normalmente quando baixado o relatório, a 1° linha ficam os títulos das colunas, e não devem ser copiados para que não haja conflitos e que o relatório fique o mais certo possível ).
Qual Coleta devo Marcar?
Todas coletas ( Essa opção serve tanto para quando tiver somente 1 coleta quento se tiver mais coletas, ela vai retornar os dados somados de todas as coletas existentes, tanto a quantidade dos Itens/Tabela/Gráfico, quanto as mensagens prontas).

Coleta individual ( A coleta individua 'Coleta 1, Coleta 2, Coleta 3, ....', serve para filtrar os(as) Itens/Tabela/Gráfico quanto as mensagens prontas apenas da coleta selecionada).

IMPORTANTE

Para garantir a integridade e precisão dos registros, siga rigorosamente esta orientação ao preencher as coletas:

Coleta 1: Nesta coleta, você é obrigado a inserir os dados necessários.

Coleta 2: A Coleta 2 é opcional, mas, caso você escolha preenchê-la, certifique-se de que ela siga a sequência após a Coleta 1.

Coleta 3: Da mesma forma, a Coleta 3 também é opcional, mas, se preenchida, deve seguir após a Coleta 2.

Por favor, assegure-se de manter a sequência rigorosa apenas para a Coleta 1. As outras coletas são opcionais, mas, se preenchidas, devem seguir a ordem.
